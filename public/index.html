Mini Brands API Go to API Playground

The supported API endpoint: https://surprise-5-api.vercel.app//api/minibrand.

  • Get the full list of Mini Brands: /api/minibrand
  • Query the API for a specific Mini Brands by series by passing the series number in the URL:
    Example: /api/minibrand/seriesno/4

API Calls & Endpoints

HTTP Method Endpoint What it does
GET /api/minibrand Returns an array of all Minibrands in our database.
GET /api/minibrand/seriesno/:seriesNo Returns an array of minibrands based on their series.
The array will contain all the minibrands in the selected series.
GET /api/minibrand/itemno/:itemNo Returns an array of minibrands based on their item Number.
The array will contain all the minibrands of the selected Item Number.
GET /api/minibrand/brandname/:brandName Returns an array of mini brands based on their brand.
GET /api/minibrand/rarity/:rarity Returns an array of mini brands based on their rarity. (Common, rare, super rare, ultra rare)
GET /api/minibrand/sf/:specialFeature Returns an array of mini brands based on their special feature. (gold, glow in the dark, metallic, rose gold, frozen moments )

Field reference

Field name Description Data type
seriesNo Series Number Number (but GR is String)
itemNo Number of Item in series Number
